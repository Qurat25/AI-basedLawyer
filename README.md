# ⚖️ AI-Based Lawyer

A web platform that provides AI-powered legal assistance. Users can ask legal questions and get intelligent, easy-to-understand responses — without needing to hire a lawyer for basic queries.

---

## 🚀 Live Demo

> Coming soon

---

## 💡 What It Does

- Users can register and log in securely
- Ask legal questions in plain English
- Get AI-generated responses based on the query
- View past conversations (v2)

---

## 🛠️ Tech Stack

**Current (v1)**
- HTML, CSS, JavaScript
- Static frontend with login & register pages

**Planned (v2 — in progress)**
- React + TypeScript (frontend)
- Node.js + Express (backend)
- MongoDB (user data & chat history)
- JWT Authentication
- OpenAI API (AI responses)
- Deployed on AWS EC2 + Vercel
